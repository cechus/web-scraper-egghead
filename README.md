# First Web scraping with python

## Requirements
```bash
$ sudo apt install python-pip
$ sudo apt install ffmpeg
$ sudo -H pip install --upgrade youtube-dl
$ pip install BeautifulSoup
$ pip install BeautifulSoup4
```
